# Geely / Belgee CX Dashboard

React-дизайн-прототип дашборда клиентского опыта для Geely и Belgee.

## Запуск

```bash
npm install
npm run dev
```

Для production-сборки:

```bash
npm run build
```

Все данные демонстрационные. Интерфейс адаптирован под десктоп, планшет и мобильные экраны.
